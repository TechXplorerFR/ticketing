# ADR - Conventions 

## Contexte

On cherche ici à définir les conventions de code utilisées dans le projet.

## Décision

Seront utilisées les conventions suivantes:

| Élément  | Convention de nommage |
| -------- | --------------------- |
| path     | kebab-case            |
| function | camelCase             |
| class    | PascalCase            |
| branches | kebab-case            |
| folder   | snake_case            |
| file     | snake_case            |

## Statut

✅ - Validé à l'unanimité

## Conséquences

Elles seront terribles et irrévocables, nous devons tout renommer. Néanmoins, tout sera propre et clair à la suite de cela.
